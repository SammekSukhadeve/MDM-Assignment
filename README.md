# MDM-Assignment
Created an Amazon Clone in My Minor Degree (MDM) Assignment for learning aout HTML and CSS in detail.

The purpose of this project was to understand:

* Structure of web pages using HTML

* Styling and layout using CSS

* Creating responsive layouts

* Using Flexbox and box model

* Designing real-world UI components

====================================================
AMAZON WEBSITE CLONE
(HTML & CSS PROJECT)
====================================================

1. WEBSITE COMPONENTS
====================================================

1) HEADER / NAVIGATION BAR
----------------------------------------------------
Elements Included:
- Amazon logo
- Location selector
- Search bar
- Sign-in option
- Returns & Orders
- Cart icon

What I Learned:
- Using <header> and <nav> tags
- Aligning items horizontally using Flexbox
- Styling input fields and buttons
- Using padding, margin, and border-radius
- Creating a professional navigation layout


2) HERO SECTION (BANNER)
----------------------------------------------------
Elements Included:
- Large background promotional image
- Overlay text
- Promotional message

What I Learned:
- Using background-image property
- background-size: cover
- Positioning using relative and absolute
- Adding overlay content on images


3) PRODUCT GRID SECTION
----------------------------------------------------
Elements Included:
- Product cards
- Product images
- Product titles
- Price details
- “See more” links

What I Learned:
- Using CSS Grid / Flexbox for layout
- Creating reusable card components
- Understanding the CSS Box Model:
  - Margin
  - Padding
  - Border
  - Content
- Adding hover effects using :hover


4) FOOTER SECTION
----------------------------------------------------
Elements Included:
- Back to top section
- Multiple columns of links
- Copyright text

What I Learned:
- Creating multi-column layout using Flexbox
- Styling dark themed sections
- Managing spacing and alignment
- Organizing links in structured format


2. HTML CONCEPTS LEARNED
====================================================
- Basic HTML document structure
- Semantic elements:
  <header>
  <nav>
  <section>
  <footer>
- Forms and input elements
- Anchor tags for navigation
- Image embedding using <img>
- Div and container structuring


3. CSS CONCEPTS LEARNED
====================================================
- CSS Selectors (class, id, element)
- Box Model
- Flexbox layout
- Text styling (font-size, color)
- Background styling
- Border and border-radius
- Hover effects
- Basic responsive design (if applied)
- Media queries (if implemented)


4. CHALLENGES FACED
====================================================
- Aligning navbar items properly
- Matching spacing similar to Amazon
- Making product cards equal in size
- Handling responsiveness
- Managing layout consistency


5. CONCLUSION
====================================================
By building this Amazon clone:

- I gained practical experience in HTML and CSS.
- I understood how real-world e-commerce websites are structured.
- I improved my layout designing skills.
- I learned how to build reusable UI components.
- I gained confidence in front-end web development.

This project helped strengthen my foundation in web development and prepared me for more advanced projects.
